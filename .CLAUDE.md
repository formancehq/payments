# Payments

## Development environment

Enter the Nix dev shell before running any commands:

```sh
nix develop
```

## OpenAPI spec generation

Source files live in `openapi/` and are merged into `openapi.yaml` at the project root.

### Full pipeline

```sh
just openapi
```

This runs, in order:

1. `compile-connector-configs` — builds connector config schemas into `openapi/v3/v3-connectors-config.yaml`
2. `compile-api-yaml` — merges all OpenAPI inputs (see `openapi/openapi-merge.json`) into `openapi.yaml`
3. `compile-api-docs` — generates API docs into `docs/api/README.md`
4. `validate-openapi` — validates the final `openapi.yaml` with `kin-openapi`

### Regenerate only the merged spec

```sh
just compile-api-yaml
```

### Regenerate SDK after spec changes

```sh
just generate-sdk   # runs openapi + speakeasy
just generate       # runs generate-sdk + go generate
```
